# yandex_backend_school

## Версия Python:
`Python 3.7.3`

## Установка:
```shell
pip install -r requirements.txt
```

## Настройка СУБД:

Из корневой папки проекта:

```shell
flask db init
flask db migrate
flask db upgrade
```

## Запуск

Из корневой папки проекта:

```shell
flask run --host=0.0.0.0 --port=8080
```
